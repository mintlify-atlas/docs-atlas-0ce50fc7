---
title: "Delete user"
description: "Permanently delete a user account from the system."
---

Permanently removes a user account. This action cannot be undone. Consider deactivating a user by setting their `Estado` to `"Inactivo"` via `updateUser` as a reversible alternative.

**Endpoint:** `USERS` service

**Action:** `deleteUser`

<Warning>
  User deletion is permanent. All session tokens for the deleted user are immediately invalidated. Activity logs referencing the user are retained for audit purposes.
</Warning>

## Request

<ParamField path="action" type="string" required>
  Must be `"deleteUser"`.
</ParamField>

<ParamField path="payload.userId" type="number" required>
  The ID of the user to delete.
</ParamField>

## Response

<ResponseField name="status" type="string" required>
  Always `"success"` on success.
</ResponseField>

## Example

```javascript
const USERS_ENDPOINT = 'https://script.google.com/macros/s/AKfycbxAqyEcAHetH6yN4qccGILL-L3IzMSPVuVJ1kpuO86GqfDXTKP8cHrrB7UkKN1r_0g5/exec';

// Show a confirmation dialog before calling this
const confirmed = confirm('Are you sure you want to delete this user? This cannot be undone.');

if (confirmed) {
  const response = await fetch(USERS_ENDPOINT, {
    method: 'POST',
    headers: { 'Content-Type': 'text/plain' },
    body: JSON.stringify({
      action: 'deleteUser',
      payload: { userId: 12 }
    }),
    redirect: 'follow'
  });

  const result = await response.json();
  if (result.status === 'success') {
    console.log('User deleted');
  }
}
```
