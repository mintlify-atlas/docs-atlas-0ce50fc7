---
title: "Change password"
description: "Update the password for a user account by ID."
---

Sets a new password for the specified user account. This action can be used by administrators to reset a user's password, or by users to change their own.

**Endpoint:** `USERS` service

**Action:** `changePassword`

<Note>
  The API does not require the current password for this action. Ensure your application enforces current-password verification at the UI layer for self-service password changes.
</Note>

## Request

<ParamField path="action" type="string" required>
  Must be `"changePassword"`.
</ParamField>

<ParamField path="payload.userId" type="number" required>
  The ID of the user whose password is being changed.
</ParamField>

<ParamField path="payload.newPassword" type="string" required>
  The new password to set for the account.
</ParamField>

## Response

<ResponseField name="status" type="string" required>
  Always `"success"` on success.
</ResponseField>

## Example

```javascript
const USERS_ENDPOINT = 'https://script.google.com/macros/s/AKfycbxAqyEcAHetH6yN4qccGILL-L3IzMSPVuVJ1kpuO86GqfDXTKP8cHrrB7UkKN1r_0g5/exec';

const response = await fetch(USERS_ENDPOINT, {
  method: 'POST',
  headers: { 'Content-Type': 'text/plain' },
  body: JSON.stringify({
    action: 'changePassword',
    payload: {
      userId: 7,
      newPassword: 'newSecurePass456'
    }
  }),
  redirect: 'follow'
});

const result = await response.json();
if (result.status === 'success') {
  console.log('Password updated successfully');
  // Optionally redirect to login
}
```
